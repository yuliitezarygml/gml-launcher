# Анализ админки в EasyCabinet

## ❌ Встроенной админки НЕТ

EasyCabinet — это **личный кабинет для игроков**, а не админ-панель.

### Что есть в EasyCabinet:

**Для обычных пользователей:**
- ✅ Регистрация (email + логин + пароль)
- ✅ Авторизация с JWT токенами
- ✅ Профиль пользователя
- ✅ Загрузка скина (PNG файл)
- ✅ Загрузка плаща (PNG файл)
- ✅ Выбор типа скина (Default/Slim)
- ✅ Переключение плащ/элитры
- ✅ Восстановление пароля через email
- ✅ Смена пароля

**API для Aurora Launcher:**
- `/aurora/auth` — авторизация
- `/aurora/join` — join сервера
- `/aurora/hasJoined` — проверка join
- `/aurora/profile` — профиль игрока
- `/aurora/profiles` — множественные профили

### ❌ Чего НЕТ (админские функции):

- Список всех пользователей
- Блокировка/разблокировка пользователей
- Удаление пользователей
- Просмотр статистики
- Управление правами
- Логи авторизаций

## Что можно сделать

### Вариант 1: Добавить админку в EasyCabinet

Создать новый модуль `admin` в backend:

```typescript
// packages/backend/src/admin/admin.controller.ts
@Controller('admin')
@UseGuards(JwtAuthGuard, AdminGuard)
export class AdminController {
  @Get('users')
  async listUsers() { ... }
  
  @Post('users/:id/block')
  async blockUser() { ... }
  
  @Delete('users/:id')
  async deleteUser() { ... }
}
```

И страницу в frontend:
```
/admin/users — список пользователей с управлением
```

### Вариант 2: Использовать оба проекта

**gml-auth (Go):**
- Админ API для управления пользователями
- Telegram бот для регистрации
- Простой и быстрый

**EasyCabinet (NestJS):**
- Веб-личный кабинет для игроков
- Загрузка скинов/плащей
- Email-уведомления

Синхронизация через общую MySQL базу или API.

### Вариант 3: Добавить веб-кабинет в gml-auth

Взять из EasyCabinet:
- Frontend страницы (React/Solid.js)
- Загрузку файлов скинов
- Email-систему

Добавить в gml-auth:
- Endpoints для загрузки скинов
- Хранение файлов в `data/skins/`
- Простой HTML интерфейс

## Рекомендация

**Для вашего проекта лучше всего:**

1. **Оставить gml-auth как основу** (у вас уже есть админка!)
2. **Добавить функции из EasyCabinet:**
   - Загрузка скинов через API
   - Простая веб-страница профиля
   - Опционально: email-уведомления

3. **Не использовать EasyCabinet полностью**, потому что:
   - Нет админки (у вас уже есть лучше)
   - Сложнее (NestJS + MySQL vs Go + JSON)
   - Для Aurora Launcher, а не GML Launcher
